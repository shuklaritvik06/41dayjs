## Regex

```
a|b => Match either a or b
a?b => Match either a or b
+ => One or more
* => Zero or more
*? => Zero or more but stop after first match
{N} => Exact N number of times
{N,M} => Range


.  => Match any character
\n => Newline character
\t => Tab character
\s => Any whitespace character
\S => Any non whitespace char
\w => Any word char (0-9), upper/lower char
\W => Any non word char (like space)
\b => word boundary
^ => start of line
$ => end of line
\\ => literal char "\"

[A-Z]
[a-z]
(..)
[asdf]
[^asdf]
```
